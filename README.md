# PurpleCowAssignment

## 프로젝트 소개
〈통통 디펜스 : 핀볼 히어로〉의 1스테이지 핵심 플레이를 참고하여 볼 발사, 몬스터 충돌, 웨이브 진행, 로그라이크식 3택지 스킬 선택 시스템을 구현했습니다.

## 개발 환경

- Unity 6000.3.10f1
- Target Platform: Android
- Language: C#

## 실행 방법

1. 제출된 APK 파일을 Android 기기에 설치합니다.
2. 게임을 실행합니다.
3. 화면을 터치/드래그하여 볼의 발사 방향을 조준합니다.
4. 몬스터를 처치하며 웨이브를 진행합니다.
5. 조건 처치 수를 달성하면 3개의 스킬 선택지가 표시됩니다.
6. 모든 웨이브를 완료하면 클리어 팝업이 표시됩니다.
7. 플레이어 체력이 0이 되면 실패 팝업이 표시되며, Retry 버튼으로 1스테이지를 재시작할 수 있습니다.

## 주요 구현 내용

### 인게임 핀볼 로직

- 볼 발사 및 물리 기반 반사 구현
- 벽 충돌 시 반사 처리
- 몬스터 충돌 시 데미지 처리
- 조준 궤적 미리보기 구현
- 모바일 터치 입력 대응

### 웨이브 시스템

- 총 20웨이브 구성
- 처치 수 조건 기반 웨이브 진행
- 웨이브 진행도 UI 구현
- 조건 달성 시 스킬 선택 UI 노출
- 마지막 웨이브 조건 달성 후 남은 몬스터를 모두 처치해야 스테이지 클리어

### 몬스터 시스템

- ScriptableObject 기반 MonsterData 관리
- 몬스터 체력, 이동속도, 그리드 크기 데이터화
- 1x1, 2x1, 1x2 몬스터 스폰 대응
- 몬스터 HP Bar 구현
- 몬스터가 FailZone에 도달하면 플레이어 HP 감소

### 로그라이크 스킬 선택 시스템

- 웨이브 조건 달성 시 3택지 스킬 선택 UI 출력
- 전체 스킬 풀에서 중복 없이 랜덤 선택지 생성
- 보유하지 않은 스킬은 Lv.1로 등장
- 이미 보유한 스킬은 레벨업 선택지로 등장
- 최대 레벨 도달 스킬은 선택지에서 제외
- 액티브 스킬 최대 4개, 패시브 스킬 최대 2개 제한 구현
- 보유 스킬 UI 구현

### 액티브 스킬

- Fire Ball
  - 타격 시 화상 중첩
  - 레벨별 지속시간, 최대 중첩 수, 초당 피해량 적용

- Ice Ball
  - 확률 기반 감속 효과
  - 레벨별 확률, 지속시간, 감속률 적용

- Laser Ball
  - 같은 행의 모든 적에게 추가 피해

- Ghost Ball
  - 적을 관통하며 피해 적용

- Cluster Ball
  - 확률적으로 특수 볼 생성
  - 생성된 특수 볼은 임시 볼로 처리되어 회수 시 재발사되지 않음

### 패시브 스킬

- Warm Heart
  - 노멀 볼 추가 피해량 증가

- Magic Mirror
  - 벽 충돌 시 다음 타격 피해량 증가

- Ruby Dagger
  - 적 전면 타격 시 치명타 확률 증가

- Emerald Dagger
  - 적 후면 타격 시 치명타 확률 증가

- Last Match
  - 적 사망 시 주변 적에게 폭발 피해

### 게임 결과 처리

- Stage Clear 팝업
- Game Over 팝업
- Retry 버튼을 통한 1스테이지 재시작
